## 2. 타이포그래피 (Typography)

> 시스템 폰트 사용 (iOS: SF Pro, Android: Roboto)
> 야외 가독성을 위해 일반 앱보다 1단계 큰 크기 적용

| 토큰 | 크기 | 굵기 | 줄간격 | 용도 |
|------|------|------|--------|------|
| `heading.xl` | 24px | 700 (Bold) | 32px | 화면 타이틀 |
| `heading.lg` | 20px | 700 (Bold) | 28px | 섹션 타이틀 |
| `heading.md` | 17px | 600 (SemiBold) | 24px | 카드 타이틀, 폼 스텝 제목 |
| `body.lg` | 16px | 400 (Regular) | 24px | 주요 본문, 리스트 항목 |
| `body.md` | 15px | 400 (Regular) | 22px | 일반 본문 |
| `body.sm` | 13px | 400 (Regular) | 18px | 보조 텍스트, 캡션 |
| `label` | 14px | 600 (SemiBold) | 20px | 폼 라벨, 버튼 텍스트 |
| `badge` | 12px | 600 (SemiBold) | 16px | 뱃지, 태그 |

## 3. 간격 (Spacing)

| 토큰 | 값 | 용도 |
|------|-----|------|
| `xs` | 4px | 아이콘-텍스트 간격, 뱃지 내부 패딩 |
| `sm` | 8px | 관련 요소 간 간격, 입력 필드 내부 패딩 |
| `md` | 16px | 카드 내부 패딩, 섹션 내 요소 간격 |

| `xl` | 32px | 큰 섹션 구분, 화면 상하 패딩 |
| `xxl` | 48px | 주요 영역 분리 |

## 5. 그림자 (Shadow)

| 토큰 | 값 | 용도 |
|------|-----|------|
| `shadow.sm` | `0 1px 2px rgba(0,0,0,0.06)` | 입력 필드 포커스 |
| `shadow.md` | `0 2px 8px rgba(0,0,0,0.1)` | 카드, 떠있는 요소 |
| `shadow.lg` | `0 4px 16px rgba(0,0,0,0.12)` | 바텀시트, 모달 |

```typescript
// RN StyleSheet 표현
shadow: {
  sm: { shadowColor: '#000', shadowOffset: { width: 0, height: 1 }, shadowOpacity: 0.06, shadowRadius: 2, elevation: 1 },
  md: { shadowColor: '#000', shadowOffset: { width: 0, height: 2 }, shadowOpacity: 0.10, shadowRadius: 8, elevation: 3 },
  lg: { shadowColor: '#000', shadowOffset: { width: 0, height: 4 }, shadowOpacity: 0.12, shadowRadius: 16, elevation: 6 },
}
```

## 6. 컴포넌트 스타일

### 6.1 버튼 (Button)

#### Primary Button (주요 액션)

```
┌─────────────────────────────┐
│         다음 단계            │  height: 52px
└─────────────────────────────┘
배경: primary.600 (#228be6)
텍스트: white, label (14px SemiBold)
모서리: 10px
pressed: primary.700 (#1c7ed6)
disabled: gray.200 배경, gray.400 텍스트
```

- 화면 하단 고정 시 좌우 `lg`(24px) 패딩, 하단 SafeArea 위
- 큰 버튼(제출, 다음): `buttonHeightLg`(56px) 사용

#### Secondary Button (보조 액션)

```
┌─────────────────────────────┐
│         임시저장              │  height: 52px
└─────────────────────────────┘
배경: primary.50 (#e7f5ff)
텍스트: primary.600 (#228be6), label
모서리: 10px
pressed: primary.100 (#d0ebff)
```

#### Outline Button (취소, 덜 중요한 액션)

```
┌─────────────────────────────┐
│           취소               │  height: 48px
└─────────────────────────────┘
배경: transparent
보더: 1.5px gray.300 (#dee2e6)
텍스트: gray.700 (#495057), label
모서리: 10px
pressed: gray.50 배경
```

#### Danger Button (삭제, 위험 액션)

```
배경: error (#fa5252)
텍스트: white
pressed: #e03131
```

#### 버튼 페어 (위저드 하단)

```
┌──────────────┐  ┌──────────────┐
│    이전       │  │    다음       │  gap: sm (8px)
└──────────────┘  └──────────────┘
  Outline (1/3)    Primary (2/3)
```

- 이전: Outline, flex: 1
- 다음: Primary, flex: 2
- 첫 스텝에서는 이전 버튼 숨김
- 마지막 스텝에서는 "다음" → "확인"

### 6.4 선택 버튼 그룹 (FormSelect)

> 드롭다운 대신 버튼 그리드로 선택. 1탭으로 선택 완료 — 야외 + 비전문가에 최적.
> 마스터코드 13그룹 모두 7개 이하 옵션이므로 버튼 그리드로 충분.

#### 옵션 3개 이하 — 1행 배치

```
  작물 상태 *
┌──────────┐ ┌──────────┐ ┌──────────┐
│   양호    │ │   보통    │ │   불량    │   height: 48px
└──────────┘ └──────────┘ └──────────┘
flex: 1 균등 분배, gap: sm (8px)
```

#### 옵션 4~7개 — 2열 그리드

```
  작물 종류 *
┌───────────────┐ ┌───────────────┐
│      벼       │ │    밭작물      │   height: 48px
└───────────────┘ └───────────────┘
┌───────────────┐ ┌───────────────┐
│     과수      │ │     채소       │
└───────────────┘ └───────────────┘
┌───────────────┐ ┌───────────────┐
│     화훼      │ │   특용작물     │
└───────────────┘ └───────────────┘
┌───────────────┐
│     기타      │
└───────────────┘
grid: 2열, gap: sm (8px)
```

#### 옵션 8개 이상 — 3열 그리드 (해당 없지만 대비)

```
grid: 3열, gap: sm (8px), 버튼 높이 44px, 폰트 13px
```

#### 스타일

```

### 6.10 토스트 메시지 (Toast)

```

error:   좌측 아이콘 error, 좌측 3px 보더 error
